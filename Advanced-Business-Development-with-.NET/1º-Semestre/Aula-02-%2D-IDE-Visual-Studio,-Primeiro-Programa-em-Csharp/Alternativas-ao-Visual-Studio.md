# VS Code
      
O VS Code é muito utilizado no mercado e é um editor de código-fonte gratuito e de código aberto desenvolvido pela Microsoft. Ele permite editar, depurar e executar códigos em diversas linguagens de programação, como JavaScript, Python, C#, Java, etc. Ele também possui uma loja de extensões que adicionam novas funcionalidades ao editor, como temas, formatação, integração com ferramentas externas, etc [18].
